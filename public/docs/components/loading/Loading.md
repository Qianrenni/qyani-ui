# Props 参数

| 参数         | 类型                                                          | 默认值         | 必填 | 说明           |
|------------|-------------------------------------------------------------|-------------|----|--------------|
| `type`     | `'breathing' \| 'spinner' \| 'skeleton'`                    | `'breathing'` | 否  | 加载动画类型       |
| `show`     | `boolean`                                                   | `true`      | 否  | 控制是否显示加载组件   |
| `text`     | `string`                                                    | `''`        | 否  | 显示在加载组件下方的文本 |

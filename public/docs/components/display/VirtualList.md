# Props 参数

| 参数           | 类型     | 默认值 | 必填 | 说明             |
|----------------|----------|--------|------|------------------|
| `data`         | `any[]`  | `[]`   | 是   | 列表数据源        |
| `itemSize`     | `number` | `50`   | 是   | 每个列表项的高度   |
| `containerHeight` | `number` | `300` | 是   | 容器可视区域高度   |

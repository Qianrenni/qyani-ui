# Props 参数

| 参数      | 类型                                                                 | 默认值         | 必填 | 说明                   |
|---------|--------------------------------------------------------------------|-------------|----|----------------------|
| `message` | `string`                                                          | `undefined` | 是  | 消息提示的文本内容            |
| `type`    | `'success' \| 'error' \| 'warning' \| 'info'`                     | `undefined` | 是  | 消息类型，影响颜色           |
| `onClose` | `() => void`                                                      | `undefined` | 否  | 点击消息时触发的关闭回调函数     |

---

# Events 事件

| 事件名   | 回调参数 | 说明             |
|-------|------|----------------|
| `close` | —    | 当消息被关闭时触发（通过点击） |

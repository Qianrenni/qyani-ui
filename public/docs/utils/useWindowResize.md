# `useWindowResize` 组合式函数

## 说明

一个用于统一管理窗口尺寸变化（`resize` 事件）的组合式工具函数。支持注册/注销处理器、自动绑定/解绑事件监听器、手动触发等能力，避免重复绑定与内存泄漏。

---

## Expose 方法

| 方法名           | 参数                                      | 返回值  | 说明                                                                 |
|----------------|-----------------------------------------|-------|----------------------------------------------------------------------|
| `addHandler`   | `(handler: (width: number, height: number) => void)` | `void` | 注册一个窗口尺寸变化处理器。首次调用时自动绑定 `resize` 事件并立即触发一次以获取当前尺寸。 |
| `removeHandler`| `(handler: (width: number, height: number) => void)` | `void` | 移除指定的处理器。当所有处理器被移除后，自动解绑 `resize` 事件监听器。                   |
| `getHandlerCount` | —                                    | `number` | 获取当前已注册的处理器数量，便于调试。                                               |
| `triggerResize`   | —                                    | `void` | 手动触发一次所有处理器的执行，使用当前窗口尺寸。常用于初始化或强制刷新。                     |
